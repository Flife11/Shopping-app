# Shopping-app

### Note:
- Hiển thị successful noti:  
  `$('#successNotificationtext').text('test')`  
  `$('#successNotificationModal').css('display', '');`
- Hiển thị failure noti:  
  `$('#failureNotificationText').text('test')`  
  `$('#failureNotificationModal').css('display', '');`

- Error code (register):
  + 401: username error
  + 402: password error
  + 403: email error
- Role:
  + client
  + admin
